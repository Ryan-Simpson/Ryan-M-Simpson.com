# Rules

- Use Server Components by default.
- Add "use client"; only for motion, hooks, browser APIs, or interactive state.
- Keep components modular.
- Avoid giant monolithic files.
- Prioritize typography, spacing, and recruiter readability.
- Avoid generic AI portfolio aesthetics.
- Avoid excessive gradients and glassmorphism.
