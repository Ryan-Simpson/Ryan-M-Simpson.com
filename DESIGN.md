# Design Direction

Premium robotics/autonomy portfolio with minimal cinematic engineering aesthetics.

# Inspired By

- Tim Wallace: whitespace, typography, restraint, storytelling
- Aakash Rajbanshi: interaction, motion, developer energy
- NVIDIA
- Waymo
- Applied Intuition
- modern robotics labs

# Theme

Dark-first, but not pure black.
Use tinted dark neutrals.

# Color

Use OKLCH colors.
Avoid #000 and #fff.
Use restrained neutrals with one technical accent.

# Typography

Strong hierarchy.
Body line length: 65–75ch.
Use clean modern sans-serif typography.
Avoid oversized generic hero text.

# Layout

Use whitespace, rhythm, and asymmetry.
Avoid repetitive card grids.
Projects should feel like technical case studies.

# Motion

Use Framer Motion sparingly.
Use staggered reveals, subtle parallax, and smooth transitions.
No bounce or distracting effects.

# Priority

Typography > spacing > hierarchy > technical storytelling > motion.
